  1. GitHub Pages 是什么？
    GitHub Pages 是由 GitHub 官方提供的一种免费的静态站点托管服务，让我们可以在 GitHub 仓库里托管和发布自己的静态网站页面，就好比是免费服务器。

hexo是快速、简洁且高效的博客框架，基于 Node.js 运行，可以将我们撰写的 Markdown 文档快速解析渲染成静态的 HTML 网页。
